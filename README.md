# Next.js project template 

This is a custom [Next.js](https://nextjs.org) project template with added configurations and packages already configured to start developing right away 

Made by [Eduardo Meza](https://eduardomeza.vercel.app)

## Getting Started

Install:

```bash
pnpm install (recommended)
# or
npm install
```

Intialize your Github repo:

```bash
git init
```

Run the development server:

```bash
pnpm dev
# or
npm run dev
```

Open [http://localhost:4000](http://localhost:4000) with your browser to see the result.

## Features

This template has turbopack enabled and uses port 4000 

It has the following packages configured:
- Typescript
- Tailwind CSS
- React Icons 
- Zustand 
- pnpm-lock.yaml

It uses the traditional layout with empty files in place:
- Header
- Footer

And it has some core folders in src and app directory already created:
- /app/actions
- /src/components
- /src/types
- /src/store
- /src/utils
- /src/hooks
- /src/cdn

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
